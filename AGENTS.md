# DATASET - Agent Knowledge Base

**Generated:** 2026-01-04 | **Commit:** 5e149cd | **Branch:** main

## OVERVIEW

Kotlin DSL library converting freeform search queries to JOOQ SQL conditions. ANTLR4 parser, type-safe field config, HTTP request integration.

## STRUCTURE

```
dataset/
├── src/main/kotlin/com/iodesystems/db/
│   ├── DataSet.kt              # Core API: DataSet { }, DataSetContext, Request/Response
│   ├── search/                 # ANTLR4 parser + condition factory
│   ├── jooq/                   # LazySql (runtime conditions)
│   └── util/                   # StringUtil (camelCase/snake_case)
├── src/main/antlr4/            # Grammar files (Lexer.g4, Parser.g4)
├── src/main/java-generated/    # ANTLR4 output (DO NOT EDIT)
├── src/test/kotlin/            # H2 in-memory tests
├── buildSrc/                   # Custom Gradle: ANTLR tasks, release automation
└── bin/release                 # Release script
```

## WHERE TO LOOK

| Task | Location | Notes |
|------|----------|-------|
| Add field config option | `DataSet.kt` → `FieldConfiguration`, `DataSetFieldConfigBuilder` | Update both + `buildDataSet()` |
| Add search syntax | `src/main/antlr4/*.g4` → regenerate → `SearchParser.kt` | Run `./gradlew generateLexer generateParser` |
| Add named search | `DataSetContext.search()` | Before `db.select()` in DSL |
| HTTP integration | `DataSet.Request.response()` / `.filter()` | response = full metadata, filter = data only |
| Runtime conditions | `DataSetContext.lazy { }` | Evaluated per-query, not at creation |
| Auto-detect fields | `Select<T>.autoDetectFields(db)` | Extension on query, adds unconfigured fields |

## CONVENTIONS

| Convention | Rule |
|------------|------|
| Indent | 2 spaces (Kotlin), 4 continuation |
| Max line | 120 chars |
| Imports | Star at 5, `java.**`, `javax.**`, `kotlin.**` order |
| Immutability | DataSet uses `copy()` - never mutate |
| Type params | `DataSet<T : Select<R>, R : Record, M>` - T=query, R=record, M=mapped |

## ANTI-PATTERNS

| DON'T | WHY |
|-------|-----|
| Edit `src/main/java-generated/` | Regenerated by ANTLR, cleaned on build |
| Use `as any` / suppress types | Library is type-safe by design |
| Mutate DataSet | Use `.copy()`, `.search()`, `.where()` |
| Skip parser regeneration after grammar change | Run `./gradlew generateLexer generateParser` |
| Commit without `./gradlew test` | H2 tests catch regressions |

## COMMANDS

```bash
./gradlew test                              # Run all tests (H2 in-memory)
./gradlew generateLexer generateParser      # Regenerate ANTLR parser
./gradlew build                             # Full build
./gradlew publishToMavenLocal -PskipSigning # Local publish (no GPG)
./bin/release <version>                     # Full release to Maven Central
```

## GOTCHAS

- **LazySql** uses deprecated JOOQ APIs - still works, monitor for breaks
- **GPG signing** required for Maven Central - use `-PskipSigning` for local
- **Configuration cache disabled** in gradle.properties (custom build logic incompatible)
- **Maven wrapper** present but dormant - project is Gradle-only
- **Field name normalization** - `autoDetectFields` uses `DataSet.fieldName()` for camelCase/snake_case

## API QUICK REF

```kotlin
// Primary entry point
val query = DataSet {
  search("daysAgo") { s -> /* custom search */ }
  db.select(
    field(USER_ID) { primaryKey(); search { f, s -> f.eq(s.toLongOrNull()) } },
    field(EMAIL) { orderable(); search { f, s -> f.containsIgnoreCase(s) } }
  ).from(USER).autoDetectFields(db)  // Optional: add unconfigured fields
}.map { record -> UserDto(...) }

// Execute
val data = query.search("john").page(0, 50).data(db).fetch()

// HTTP
val response = DataSet.Request(search = "john", showCounts = true).response(db, query)
```
